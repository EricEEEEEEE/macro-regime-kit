# Roadmap

## v0.1

- FRED macro regime monitor
- Three-act regime probabilities
- CPI YoY calculation
- State/history JSON output
- Obsidian weekly report sync
- Portfolio transmission from `portfolio_risk.json`
- Thesis feedback dry-run/apply block
- Codex/Claude skill export
- CI, packaging, and secret scan

## v0.2

- PyPI release
- Better Obsidian templates
- ChartsView optional rendering
- JSON schema validation
- Example launchd and cron templates
- More detailed portfolio-risk schema docs

## v0.3

- Read-only adapter examples for:
  - Interactive Brokers Flex
  - Binance account balances
  - Public wallet snapshots
- Adapter plugin interface
- More robust thesis-document targeting

## Explicit Non-Goals

- No trading automation
- No wallet signing
- No withdrawal support
- No private portfolio data in examples
