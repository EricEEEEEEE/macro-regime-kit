# Obsidian Workflow

1. Confirm `obsidian.vault_path`.
2. Confirm the macro folder exists.
3. Run `macro-regime weekly`.
4. Run `macro-regime obsidian-sync`.
5. Check copied Markdown in the configured macro folder.

For thesis feedback:

1. Run `macro-regime thesis-feedback`.
2. Review changed files.
3. Apply only with `macro-regime thesis-feedback --apply`.

Do not run apply without user approval.
